What this is: The system of record for every recurring visual and audio element. Characters, locations, the grade string, music beds, and card styles all live here. Each asset goes through a propose → approve → retire gate — nothing is used in content creation until you approve it. Approved assets are locked; changes require a new version through the same gate.

These assets are referenced automatically when content is created: the grade string is injected into every image prompt, character refs condition face consistency, location refs anchor scene plates.
